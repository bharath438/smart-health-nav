# HealAI · Universal Health Navigator

AI-powered health navigator for humans and animals in India. Symptom guidance, cost estimates, government schemes, Ayurveda & modern medicine, and emergency help.

## Tech Stack

- **React** (Vite)
- **Tailwind CSS v4**
- **Vitest** for testing

## Features

- Hero section with animated badge and health score card
- Quick action cards: AI Chat, Cost Estimator, Hospitals, Govt Schemes, Image Check, Health Records, Ayurveda, Farmer Mode
- Wellness tips and seasonal alerts
- Human Mode section with Ayurveda-aware guidance
- Fully responsive design
- Emergency 108 button

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Test

```bash
npx vitest run
```
